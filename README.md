# EventsRunner

This project is still under development. It is not yet ready for use. Thank you for your interest.

Have a look at the kubernetes sensor which will be used to send kubernetes events to the events-runner and is maintained under the[eventsrunner-k8s-sensor](https://github.com/luqmanMohammed/eventsrunner-k8s-sensor) project. The sensor on-itself can be used to do limited event-driven automation, follow repository README for more info. Eventsrunner-k8s-sensor is production ready.
